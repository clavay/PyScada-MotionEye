PyScada MotionEye Extension
========================

This is a extension for PyScada to support read and write to MotionEye devices.


What is Working
---------------

 - nothing is test


What is not Working/Missing
---------------------------

 - Test with real hardware
 - Documentation

Installation
------------

 - pip install pyscada-motioneye
